# Autentication_Security
Autentication com SpringSecurity

Aplifcação simples para controle de cadastro web de vagas de emprego com armaenamento em BD.

<h4>Tecnologias utilizadas:
<li> Java 11
<li> Spring Security
<li> Spring Web
<li> Spring Data JPA
<li> Maven
<li> Lombok
<li> H2
